## Requirements
- python > 3.9
- pyenv (Optional)
- Docker (Optional)

## Setup

1. Create/activate virtualenv:
    ```
    pyenv activate venv
    ```

1. `pip install -r requirements.txt`

## Running (Development)

1. `python main.py`

## Testing

1. Navigate to `http://<ip>:8000/docs` for the FastAPI application docs page
